# DeepFace Emotion Analyzer

A deep learning project for facial emotion recognition using the FER2013 dataset.

## Project Overview
This project builds a deep learning model to classify facial emotions from images.

## Tools Used
- Python
- TensorFlow / Keras
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Files
- `deepfaceemotion-2.py`: Python script
- `DeepFaceEmotion.ipynb`: Jupyter Notebook
- `DeepFaceEmotion_Report.pdf`: Final report
- `DeepFaceEmotion_Presentation.pptx`: Presentation

## Results
The model was evaluated using accuracy/loss curves and classification results.
